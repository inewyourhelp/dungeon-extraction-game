## 🌍 **世界观背景设定**

### 📖 **核心故事线**

```
在一个名为"埃尔德利亚"(Eldoria)的中世纪奇幻世界中，
古代文明曾拥有强大的魔法力量，但因过度使用导致灾难性后果。

如今，世界处于"大崩坏"后的重建期：
- 古代遗迹散落各地，隐藏着失落的知识
- 怪物因魔力泄漏而变异，威胁人类生存
- NPC 们各自为战，试图恢复世界的平衡

玩家扮演一名无名冒险者，通过探索地牢、击败怪物、
收集资源，逐步揭开世界真相，最终决定是否重启古代魔法。
```

## 🎨 **NPC 对话场景配色规范**

### ✅ **关系值变化时的背景色调**

| NPC 关系状态 | 对话框背景色 | HEX 代码 | 情感表达 |
|--------------|--------------|----------|----------|
| **中立** | ⚫ 深灰半透明 | rgba(60,60,60,0.8) | 标准商务感 |
| **友好** | 🟢 绿色渐变 | #4CAF50→#66BB6A (线性渐变) | 温暖、信任 |
| **敌对** | 🔴 红色半透明 | rgba(244,67,54,0.8) | 紧张、危险 |
| **信赖** | 💛 金色光晕 | #FFD700 (带发光效果) | 珍贵、特殊 |

### ✅ **关键剧情对话的特殊处理**

```markdown
触发条件：主线任务关键节点 + NPC 关系≥50

配色方案:
├─ 背景：深蓝色半透明 rgba(33,150,243,0.7)
├─ 边框：金色发光 #FFD700 (动态脉冲)
└─ 文字：白色带阴影 #FFFFFF + text-shadow

效果描述:
- 对话框边缘金色光晕呼吸灯效果
- NPC 头像周围蓝色魔法粒子环绕
```
