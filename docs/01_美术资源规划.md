## 📋 设计背景

### 游戏类型
- **2D 像素风** + **魔法工业世界观** + **地牢搜打撤**
- 风格要求：**稳重战士感** + **地牢紧张感**

### 核心美术方向
```
西方魔幻 × 魔法工业融合
├─ 稳重战士：金属质感、厚重线条、实用主义
└─ 地牢紧张：暗色调对比、危险氛围、资源稀缺感
```

## 🎯 第三部分：元素附着层数 UI 图标设计

### 📐 **设计规范**

#### 尺寸标准：
- **敌人头顶图标**: 24x24px（主图标）+ 8px（数字标签）
- **UI 背包图标**: 32x32px

#### 布局结构：
```
┌──────────────┐
│   [元素图标] │ ← 24x24px / 32x32px
│      +N      │ ← 数字标签（白色粗体）
│  ━━━━━━━━━   │ ← 倒计时进度条
└──────────────┘
```

### 🎬 **动态效果规范**

#### 层数叠加动画：
```
初始状态: [图标] +1 → 淡入 (0.3s)
叠加时:    [图标] 闪烁 + 数字增加（0.2s）
持续中:   [图标] 微幅呼吸动画（每 2s 一次）
即将消失：[图标] 快速闪烁（最后 1s）
```

#### 敌人头顶显示规则：
- **最多同时显示**: 3 个不同元素图标
- **层数超过 3**: 显示"+3"，后续层数用数字累加
- **优先级**: 火 > 电 > 水 > 草 > 冰 > 暗 > 圣
