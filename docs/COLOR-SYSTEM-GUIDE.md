## 🎨 **色彩使用禁忌**

### ❌ **避免使用的配色组合**

| 错误示例 | 问题描述 | 正确做法 |
|----------|----------|----------|
| 🔴 红 + 🟢 绿 | 对比度过强，视觉疲劳 | 改为 🔵蓝 + 🟠橙 |
| ⚫ 黑 + ⚪ 白 (大面积) | 缺乏层次感 | 加入灰色/棕色过渡 |
| 💜 紫 + 🔴 红 | 色调冲突，不和谐 | 改为💜紫 + 🔵蓝 |

### ✅ **推荐配色原则**

1. **60-30-10 法则**: 主色 60%, 辅助色 30%, 点缀色 10%
2. **对比度平衡**: 文字与背景对比度≥4.5:1 (WCAG AA)
3. **情感匹配**: 危险场景用冷色调，安全区域用暖色调
