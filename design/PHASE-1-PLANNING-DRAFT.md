## 🔹 一、核心玩法机制

### 1.1 移动控制方式
- **输入设备**: 键盘 WASD / 方向键
- **移动维度**: 2D 平面（X/Y 轴）
- **控制类型**: 
  - [ ] 直接速度控制（按键=即时速度）
  - [ ] 加速度控制（按键=加速，松开=减速）← **推荐**

### 1.2 移动模式
- **基础移动**: 上下左右四个方向
- **斜向移动**: WASD 同时按下时的合成速度
- **速度限制**: 
  - 最大移动速度：? px/s
  - 最小移动速度：0 px/s

## 🔹 二、操作手感要求

### 2.1 物理参数（待确认）

| 参数 | 建议值 | 说明 |
|------|--------|------|
| **最大速度** | 200-300 px/s | 角色移动上限 |
| **加速度** | 800-1200 px/s² | 从静止到最大速度的快慢 |
| **减速度/摩擦力** | 600-1000 px/s² | 松开按键后的减速效果 |
| **转向响应时间** | < 0.1s | 方向切换的延迟 |

### 2.2 手感特征

#### ✅ 期望体验
- [x] **流畅平滑**: 不是生硬的瞬间启动/停止
- [x] **有重量感**: 移动时有加速度和减速过程
- [x] **响应及时**: 输入与反馈无明显延迟
- [ ] **惯性效果**: 是否保留轻微惯性？

#### ❌ 避免体验
- [ ] 按键即瞬间达到最大速度（过于生硬）
- [ ] 松开后立刻停止（缺乏真实感）
- [ ] 斜向移动速度过快（需要归一化处理）

## 🔹 三、碰撞检测规则

### 3.1 与墙壁/障碍物的碰撞

#### 检测方法
- **主要方案**: CharacterBody2D + CollisionShape2D ← **Godot 4.x 推荐**
- **辅助工具**: RayCast2D（用于前方障碍物预判）

#### 碰撞响应
- [x] **停止移动**: 碰到墙壁时立即停止
- [ ] **滑动效果**: 沿墙面滑动的可能性？
- [ ] **反弹效果**: 是否需要弹回？

### 3.2 支持的地形类型

| 地形 | 是否支持 | 说明 |
|------|----------|------|
| 垂直墙壁 | ✅ | 基础功能 |
| 斜坡/台阶 | ⏸️ | Phase 2 考虑 |
| 可破坏墙壁 | ⏸️ | Phase 3 考虑 |

### 3.3 碰撞层级（Collision Layer/Mask）

```
Player Collision Layers:
- Layer 1: 地面/Walls (静态障碍物)

- Layer 3: Items/Collectibles (可拾取物) - 未来扩展
```

## 🔹 四、动画状态机设计

### 4.1 基础动画状态

| 状态 | 说明 | 触发条件 |
|------|------|----------|
| **Idle (站立)** | 静止状态 | 速度 = 0 |
| **Walk (行走)** | 移动中 | 速度 > 0 |
| **Turn (转向)** | 方向切换瞬间 | 方向改变时短暂播放 |

### 4.2 动画过渡规则

- [x] **只在状态变化时重播**: 避免卡顿
- [ ] **混合时间**: 状态切换的平滑过渡时长？(0.1s)
- [ ] **方向镜像**: 左右移动是否需要镜像动画？

### 4.3 动画资源需求

| 动画 | 帧数 | 备注 |
|------|------|------|
| Idle (站立) | 2-4 帧 | 呼吸效果 |
| Walk (行走) | 6-8 帧 | 左右各一套 |
| Turn (转向) | 3-5 帧 | 可选，可简化为方向切换 |

## 🔹 六、技术实现要点

### 6.1 推荐技术方案

```gdscript
# Player Controller (player_controller.gd)
extends CharacterBody2D

# 物理参数
@export var max_speed: float = 300.0
@export var acceleration: float = 1000.0
@export var deceleration: float = 800.0

# 输入处理
var input_vector := Vector2.ZERO

func _physics_process(delta):
    # 读取输入
    input_vector = Vector2(
        Input.get_action_strength("move_right") - Input.get_action_strength("move_left"),
        Input.get_action_strength("move_down") - Input.get_action_strength("move_up")
    ).normalized()
    
    # 应用加速度
    velocity = velocity.move_toward(input_vector * max_speed, acceleration * delta)
    
    # 移动
    move_and_slide()
```
