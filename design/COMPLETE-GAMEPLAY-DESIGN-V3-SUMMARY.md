## 🔥 五大核心机制速览

### 1️⃣ 增援范围机制 (核心亮点！独特卖点！)
- **半径**: 64px (可调 32~128px)
- **最大数量**: 3 个增援敌人/次
- **延迟触发**: 0.5s 后逐个加入战斗队列 (FIFO)

### 🔥 2️⃣ **元素反应系统** ← **核心机制已更新！**

#### 七大基础反应类型

| 反应名称 | 触发条件 | 效果描述 | 伤害倍率 |
|----------|----------|----------|----------|
| **蒸发 (Vaporize)** | 水 + 火 / 火 + 水 | 元素能量剧烈冲突，双倍伤害 | ×2.0x |
| **融化 (Melt)** | 冰 + 火 / 火 + 冰 | 热能/冷能爆发，范围伤害 | ×1.5x |
| **感电 (Overload)** | 电 + 水 / 水 + 电 | 电流通过水体传导，连锁伤害 | ×1.2x/s |
| **燃烧 (Burning)** | 草 + 火 / 火 + 草 | 火焰在植物上蔓延，持续伤害 | ×0.8x/s |
| **冻结 (Freeze)** | 水 + 冰 / 冰 + 水 | 目标被完全冻结，无法行动 | - |
| **超载 (Charged)** | 电 + 火 | 爆炸造成范围伤害并击退敌人 | ×1.3x |
| **绽放 (Bloom)** | 草 + 水 | 生成种子，可触发额外反应 | ×0.5x |

#### 元素附着机制速查

```
攻击 → 附加元素 A(1~3 层)
    ↓
再受元素 B 影响 → 双元素共存
    ↓
自动触发反应 (消耗各 1 层)
    ↓
剩余层数继续积累 → 可再次反应！
```

**战术组合推荐:**
- **快速击杀**: 火 + 水 → ×2.0x 瞬间伤害 ⭐⭐⭐
- **AOE 清场**: 电 + 水 → 连锁伤害 ⭐⭐
- **控制敌人**: 冰 + 水 → 完全定身 1~2s ⭐⭐
