## 📋 核心机制变更说明

### 🔥 关键更新：从"属性克制"改为"元素反应系统"

**原设计:** 5 种元素相互克制，数值加成 (1.5x/0.75x)  
**新设计:** 双元素共存自动触发特殊效果，强调战术配合！

**核心机制:**
- **元素附着**: 每次攻击附加 1~3 层对应元素
- **反应触发**: 两种不同元素同时存在 → 自动触发
- **消耗机制**: 反应后消耗附着层数，可重新积累

**七大基础反应:**
1. **蒸发 (Vaporize)**: 水 + 火 → ×2.0x 瞬间伤害
2. **融化 (Melt)**: 冰 + 火 → ×1.5x 范围伤害
3. **感电 (Overload)**: 电 + 水 → ×1.2x/s 连锁伤害
4. **燃烧 (Burning)**: 草 + 火 → ×0.8x/s 持续伤害
5. **冻结 (Freeze)**: 水 + 冰 → 完全定身 1~2s
6. **超载 (Charged)**: 电 + 火 → ×1.3x 击退效果
7. **绽放 (Bloom)**: 草 + 水 → 生成种子，可触发额外反应

## ⚠️ 风险评估与预案 (待团队评审)

### 风险 1: 元素反应系统复杂度超预期
**概率:** 高  
**影响:** 中  

**预案:** 
- **简化方案:** Phase 1 只实现前 4 种基础反应，后续迭代添加剩余 3 种
- **测试策略:** 优先验证核心机制 (附着/触发),再优化数值平衡

---

### 风险 2: UI 反馈不清晰
**概率:** 中  
**影响:** 低  

**预案:** 
- **视觉提示:** 元素附着层数用图标显示在敌人头顶
- **特效强化:** 反应触发时添加明显动画和音效
