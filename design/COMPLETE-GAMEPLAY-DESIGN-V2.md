## 三、回合制战斗 ← **已更新元素反应系统！**

### 🔥 核心机制变更:从"属性克制"改为"元素反应"

> **设计理念:** 通过不同元素的组合触发特殊效果，而非简单的数值加成！

### 3.2 五大基础元素

| 元素 | 代表 | 特性 |
|------|------|------|
| **Fire (火)** | 火焰、熔岩 | 持续伤害、点燃 |
| **Water (水)** | 水流、冰霜 | 冷却、冻结 |
| **Electric (电)** | 雷电、电流 | 快速打击、连锁 |
| **Grass (草)** | 植物、藤蔓 | 持续伤害、缠绕 |
| **Ice (冰)** | 寒冰、霜冻 | 减速、冻结 |

### 🔥 3.3 元素反应系统 (核心战斗机制)

#### 3.3.1 七大基础反应类型

| 反应名称 | 触发条件 | 效果描述 | 伤害倍率 |
|----------|----------|----------|----------|
| **蒸发 (Vaporize)** | 水 + 火 / 火 + 水 | 元素能量剧烈冲突，双倍伤害 | ×2.0x |
| **融化 (Melt)** | 冰 + 火 / 火 + 冰 | 热能/冷能爆发，范围伤害 | ×1.5x |
| **感电 (Overload)** | 电 + 水 / 水 + 电 | 电流通过水体传导，连锁伤害 | ×1.2x/s |
| **燃烧 (Burning)** | 草 + 火 / 火 + 草 | 火焰在植物上蔓延，持续伤害 | ×0.8x/s |
| **冻结 (Freeze)** | 水 + 冰 / 冰 + 水 | 目标被完全冻结，无法行动 | - |
| **超载 (Charged)** | 电 + 火 | 爆炸造成范围伤害并击退敌人 | ×1.3x |
| **绽放 (Bloom)** | 草 + 水 | 生成种子，可触发额外反应 | ×0.5x |

#### 3.3.2 元素附着机制

**核心规则:**
- 每次攻击会为目标附加对应元素的"附着层数"(1~3 层)
- 当两种不同元素同时存在于同一目标时 → **自动触发反应**
- 反应后消耗附着层数，可重新积累

**战斗流程示例:**
```
1. 战士使用火球术 → 敌人附着 Fire(2 层)
2. 法师使用水枪 → 触发蒸发，造成×2.0x 伤害
3. Fire 和 Water 各消耗 1 层
4. 剩余：Fire(1), Water(1)
5. 再次攻击可继续触发反应！
```

#### 3.3.3 战术价值

| 组合 | 适用场景 | 风险等级 |
|------|----------|----------|
| **火 + 水 (蒸发)** | 快速击杀单体目标 | ⭐ 低 |
| **电 + 水 (感电)** | AOE 清场，连锁伤害 | ⭐⭐ 中 |
| **冰 + 水 (冻结)** | 控制敌人，完全定身 | ⭐⭐ 中 |
| **草 + 火 (燃烧)** | 持续消耗，持久战 | ⭐⭐ 低 |

### 3.4 AI 行为逻辑 (怪物)

| AI 类型 | 特征 | 策略 |
|---------|------|------|
| **Aggressive** | 优先攻击低血量目标 | 击杀威胁最大者 |
| **Tactical** | 根据元素反应选择技能 | 最大化反应伤害 |
