## 📦 **六大模块定义**

### 【模块一：大世界系统】
- **核心定位:** 玩家自由探索的主舞台，连接所有其他系统的枢纽
- **包含内容:** 世界结构类型、节点分布方式与密度、角色移动机制

### 【模块二：领地建设系统】
- **核心定位:** 玩家的**安全区与资源中心**,提供持久化进度和角色成长
- **包含内容:** 基地功能定位、建筑类型分类、升级体系框架

### 【模块三：固定节点系统】
- **核心定位:** 地图上**永久存在的关键地点**,提供稳定的挑战与奖励
- **包含内容:** 节点类型划分、难度分级逻辑、奖励池构成

### 【模块四：随机节点系统】
- **核心定位:** 地图上**动态生成的临时事件点**,增加探索的不确定性与趣味性
- **包含内容:** 生成规则框架、发现条件类型、刷新机制

### 【模块五：Roguelike 地牢】
- **核心定位:** **单次探索的完整流程**,高风险高回报的核心玩法
- **包含内容:** 探索流程、房间类型分类、战斗模式选择、战利品体系框架、撤离与死亡机制

### 【模块六：剧情演绎系统】
- **核心定位:** 贯穿所有模块的**叙事与角色关系系统**,增强沉浸感
- **包含内容:** NPC 分类（大世界/领地/地牢三层）、对话树结构设计、好感度体系框架

## 🔄 **模块间数据流总览**

### 【资源流动】:
```
大世界采集 → 领地建设系统 (建筑升级材料)
Roguelike 地牢战利品 → 背包系统 / 领地建设系统
固定节点奖励 → 背包系统
随机节点产出 → 背包系统 / 剧情演绎系统
```

### 【状态传递】:
```
角色等级/装备 → 大世界移动速度、战斗能力
建筑等级 → 解锁功能、强化效果
好感度 → 对话内容、交易折扣
```

### 【进度持久化】:
- **永久保存:** 领地建设、背包物品、好感度
- **条件保存:** Roguelike 战利品（成功撤离后永久，死亡可能丢失）

## 📊 **模块边界划分总结**

| 模块 | 核心职责 | 明确分界 |
|------|----------|----------|
| **大世界系统** | 自由探索的主舞台、连接枢纽 | ❌ 不包含战斗/建筑/NPC 对话等具体玩法 |
| **领地建设系统** | 安全区、资源中心、角色成长 | ❌ 不包含战斗、不处理地牢流程 |
| **固定节点系统** | 永久存在的关键地点入口 | ❌ 不包含内部探索（交给 Roguelike 地牢） |
| **随机节点系统** | 动态生成的临时事件点 | ❌ 不包含复杂流程（简单交互或进入地牢） |
| **Roguelike 地牢** | 单次探索的完整高风险流程 | ❌ 不包含永久进度、不处理领地建设 |
| **剧情演绎系统** | 叙事与角色关系（贯穿所有模块） | ✅ 跨模块存在，但独立于其他系统的核心逻辑 |
