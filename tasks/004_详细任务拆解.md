## 🎯 第一阶段：核心玩法设计 (预计 2 周)

### T001 - 回合制战斗系统设计
**负责人：** 系统策划 agent  
**任务内容：**
- [ ] 定义回合制战斗流程（玩家回合→敌人回合→结算）
- [ ] 卡牌使用规则（消耗、冷却、目标选择）

- [ ] 状态效果系统（ Buff/Debuff）

**输出文档：** `docs/01_战斗系统设计.md`

### T002 - 卡牌系统设计
**负责人：** 系统策划 + 数值策划 agent  
**任务内容：**

- [ ] 卡牌属性定义（名称、描述、效果、稀有度）
- [ ] 还原机制设计（还原消耗、冷却时间）
- [ ] 特殊功能实现（魔法效果、道具效果）

**输出文档：** `docs/02_卡牌系统设计.md`  
**输出文件：** `config/cards/card_templates.yaml` (模板)

### T003 - 地下城生成机制设计
**负责人：** 关卡策划 agent

- [ ] 转化逻辑（如何转化为卡牌？）
- [ ] 地下城难度分级

**输出文档：** `docs/03_地下城系统设计.md`
