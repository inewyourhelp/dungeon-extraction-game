## 🎯 任务目标
建立游戏主角的移动控制和碰撞检测，这是所有后续玩法的基础。

## 📝 需求详情

### 1. 角色控制
- [ ] 实现 WASD/方向键控制英雄移动
- [ ] 支持平滑移动（不是格子跳跃）
- [ ] 添加简单的动画状态（站立、行走）

### 2. 碰撞系统
- [ ] 主角与地牢墙壁的碰撞检测
- [ ] 使用 TileMap 进行碰撞查询
- [ ] 防止穿墙和越界

### 3. 技术栈要求
- Godot 4.x + GDScript
- 遵循项目命名规范（snake_case）
- 像素尺寸：16x16 / 32x32

## 📁 建议目录结构
```
scripts/core/

└── collision_system.gd     # 碰撞系统工具类
```
