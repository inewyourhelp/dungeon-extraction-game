# 🎨 世界观设定 (已确认)

## 📅 2026-03-16 - 最终版

### 🌍 核心背景
**魔法与斗气并存的世界**

---

### 👑 上古传说

#### **白魔导师**
- **地位：** 当世第一人，无比强大
- **特点：** 酷爱研究
- **成就：** 
  - 创造了魔族（纯魔力构成的可移动生物）
  - 创造了地下城（纯魔力构成的虫洞空间）
- **结局：** 封印所有地下城后失踪/牺牲

---

### 🧛 **魔族设定**

| 特性 | 描述 |
|------|------|
| **构成** | 完全由魔力构成 |
| **生存限制** | 天生无法长时间待在正常世界 |
| **居住地** | 只能生活在地下城中 |
| **魔王循环** | 每10年诞生一只魔王 → 实力极强 → 活10年后衰弱成为智者 |

---

### 🏰 **地下城设定**

| 特性 | 描述 |
|------|------|
| **本质** | 虫洞空间，纯魔力构成 |
| **功能** | 可吸收正常世界的物品/建筑转化为卡牌 |
| **数量** | 仅几个（被封印） |
| **当前状态** | 500年后封印破碎，重新开启 |

---

### 🃏 **卡牌系统**

| 特性 | 描述 |
|------|------|
| **来源** | 地下城吸收正常世界物品/建筑转化而来 |
| **功能1** | 可还原成原物品 |
| **功能2** | 具有特殊功能（魔法效果） |
| **战斗方式** | **回合制卡牌对战** ⚔️🎴 |

---

### 📜 **历史时间线**

```
上古时代：
├── 白魔导师创造魔族与地下城
└── 魔族开始入侵正常世界

第1次魔灾：
├── 魔王发起进攻
└── 各族联合抵抗

第2次魔灾：
├── 魔王再次降临
└── 战斗持续...

第3次魔灾：
├── 魔王最终攻势
└── **白魔导师封印所有地下城** (仅几个)
    └── 和平时期开始

500年后：
├── 封印破碎
└── **魔灾再次来临** (故事开始)
```

---

### 🎮 **游戏核心循环**

1. **探索地下城** → 吸收物品/建筑转化为卡牌
2. **回合制战斗** → 使用卡牌对抗魔族
3. **基地建设** → 发展据点，抵抗魔灾
4. **收集卡牌** → 还原物品/使用特殊功能
5. **魔王挑战** → 每10年魔王降临（Rogue元素）
