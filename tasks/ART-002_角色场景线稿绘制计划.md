## 🎨 **第一部分：角色设计稿**

### 👤 **1. 主角 (Player Hero)** - 核心角色

#### 📐 **尺寸标准:**
- **默认**: 32x32px (经典像素比例)
- **战斗姿态**: 48x32px (横向展示)

#### 🎨 **视觉特征:**
```
风格: "稳重战士感" + "地牢紧张感"
材质: 金属质感、厚重线条、实用主义
色调: 深灰/铁锈红为主，点缀元素色彩
```

#### 👕 **服装细节:**
| 部位 | 设计要点 | 像素尺寸 |
|------|---------|---------|
| **头盔** | 带护颈的重型骑士盔，有斗气纹路 | 16x12px |
| **铠甲** | 板甲 + 锁子甲混合，金属反光 | 32x20px |
| **武器** | 单手剑/战锤 (根据职业) | 8x16px |
| **披风** | 破损的深灰色斗篷，随风飘动 | 4x12px |

#### 🎨 **色彩方案:**
```
主色: #5D4037 (深褐 - 皮革/布料)
辅色: #616A6B (灰绿 - 石墙背景对比)
金属: #9E9E9E (银灰 - 铠甲反光)
点缀: #FF5722 (橙红 - 斗气纹路发光)
```

#### 🎬 **动画帧数:**
| 动作 | 帧数 | 说明 |
|------|------|------|
| **待机** | 4 帧 | 呼吸 + 轻微晃动 |
| **行走** | 8 帧 | 步态稳健，金属碰撞声 |
| **攻击** | 6 帧 | 挥剑/锤击动作 |
| **受伤** | 3 帧 | 闪烁红光后恢复 |

### 👹 **2. 敌人 (Enemies)** - 地牢威胁

#### 📐 **尺寸标准:**
- **小怪**: 16x16px / 24x24px
- **精英**: 32x32px
- **Boss**: 48x48px (最终决战)

#### 👹 **敌人类型设计:**

##### 🔸 **A. 地牢守卫 (Dungeon Guardian)** - Lv.1~5
```
外观: 骷髅士兵/石像鬼
材质: 白骨/岩石质感，无金属
色调: #BDBDBD (灰白) + #616161 (阴影)

特征:
- 眼眶发光 (#FF5722 - 火元素)
- 手持生锈武器
- 移动缓慢但力量大
```

##### 🔸 **B. 魔法学徒 (Magic Apprentice)** - Lv.3~8
```
外观: 蓝袍法师/女祭司
材质: 布料 + 魔法饰物
色调: #212121 (深黑) + #00BCD4 (青蓝符文)

特征:
- 手持法杖/水晶球
- 漂浮在空中
- 施法时环绕魔法光圈
```

##### 🔸 **C. 元素生物 (Elemental Beast)** - Lv.5~12
```
外观: 火焰史莱姆/冰霜巨像
材质: 纯元素构成，无实体
色调: 根据元素类型变化

特征:
- 火系：橙红渐变 + 粒子效果
- 水系：青蓝流动 + 水滴溅射
- 电系：金黄闪烁 + 电弧跳跃
```

##### 🔸 **D. Boss 级敌人 (Boss)** - Lv.10+
```
外观: 深渊领主/远古巨龙
材质: 混合金属/元素/魔法
色调: #2C3E50 (深灰蓝) + #F39C12 (暗金符文)

特征:
- 体型巨大 (48x48px)
- 特殊攻击动画
- 血条显示在头顶
```

## 🏰 **第二部分：场景线稿**

### 🏰 **1. 基地建筑 (6 种风格)**

#### 📐 **尺寸标准:**
- **主堡**: 32x48px (最高)
- **仓库**: 40x28px (最宽)
- **瞭望塔**: 16x40px (高瘦型)
- **其他建筑**: 24~32x32px

#### 🏰 **A. 主堡 (Main Keep)** - 基地核心
```
形态: 中央塔楼 + 四角防御塔
材质: 深灰石材 + 铁甲门 + 魔法符文墙
色调: #2C3E50 (深灰蓝) + #F39C12 (暗金符文)

细节:
- 塔楼顶部飘扬的旗帜（可更换颜色）
- 大门上的公会徽章浮雕
- 夜间发光的魔法符文窗格
- 防御塔的箭孔和炮口

等级变体:
Lv.1: 基础石墙，无装饰
Lv.2: 添加铁甲门环
Lv.3: 符文窗格发光
Lv.4+: 塔顶旗帜 + 魔法护盾光晕
```

#### ⚔️ **B. 兵营 (Barracks)** - 战士训练地
```
形态: 长条形单层建筑，带训练场围栏
材质: 砖石 + 木质横梁 + 武器挂架
色调: #5D4037 (深褐) + #616A6B (灰绿)

细节:
- 门口悬挂的兵器挂架（剑/盾/矛）
- 训练场中的靶子/假人
- 屋顶的瞭望哨位
- 夜晚营火照明效果

等级变体:
Lv.1: 基础草棚结构
Lv.2: 砖石墙体 + 木门
Lv.3: 武器挂架 + 训练场完善
Lv.4+: 瞭望塔 + 夜间营火
```

#### 🔨 **C. 铁匠铺 (Blacksmith)** - 装备锻造
```
形态: 开放式工坊，带熔炉和锻打台
材质: 砖石 + 金属管道 + 魔法熔炉
色调: #3E2723 (深褐) + #FF5722 (橙红)

细节:
- 中央熔炉（持续发光/冒烟）
- 墙壁上的工具挂架（锤/钳/砧）
- 屋顶排烟管道（魔法工业特征）
- 锻造时飞溅的火花粒子效果

等级变体:
Lv.1: 简易熔炉，无工具墙
Lv.2: 完善工具挂架 + 排烟管道
Lv.3: 魔法熔炉符文发光
Lv.4+: 自动化锻造机械臂
```

#### 🔬 **D. 炼金塔 (Alchemy Tower)** - 药剂研发
```
形态: 圆柱形高塔，带玻璃球体和管道系统
材质: 透明玻璃 + 魔法金属支架 + 发光容器
色调: #212121 (深黑) + #00BCD4 (青蓝)

细节:
- 塔顶旋转的玻璃观测球（内部有悬浮药剂）
- 外墙附着的魔法管道系统
- 窗口内发光的实验容器
- 夜间整体发光效果（紫色/蓝色交替）

等级变体:
Lv.1: 基础石塔，无玻璃球
Lv.2: 添加管道系统 + 实验容器
Lv.3: 顶部观测球发光
Lv.4+: 全塔魔法护盾 + 能量流动特效
```

#### 📦 **E. 仓库 (Warehouse)** - 资源存储
```
形态: 低矮长条形，多门设计
材质: 强化木板 + 金属加固条 + 锁扣系统
色调: #4E342E (深棕) + #616161 (灰黑)

细节:
- 多扇带铁环的大门（可动画开关）
- 屋顶的货物堆放区（木箱/麻袋堆叠）
- 门口的守卫哨位
- 夜间外部照明灯（火把/灯笼）

等级变体:
Lv.1: 简易木板房，无加固条
Lv.2: 金属加固 + 多门设计
Lv.3: 守卫哨位 + 外部照明
Lv.4+: 魔法封印符文 + 自动防御机关
```

#### 👁️ **F. 瞭望塔 (Watchtower)** - 警戒侦查
```
形态: 细长塔楼，顶部开放平台
材质: 石砖 + 木质栏杆 + 望远镜装置
色调: #37474F (深青灰) + #FFD700 (金黄)

细节:
- 顶部旋转的瞭望望远镜装置
- 塔身外部的攀爬梯/绳索
- 夜间探照灯（可旋转光束）
- 旗杆上的警戒旗帜（随风飘动）

等级变体:
Lv.1: 简易木塔，无望远镜
Lv.2: 石砖结构 + 瞭望平台
Lv.3: 望远镜装置 + 探照灯
Lv.4+: 魔法预警符文圈（遇敌发光）
```
