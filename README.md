# Dungeon Extraction (地下城也要搜打撤)

## 🎮 游戏类型
**2D 像素 + 地下城 + 建设 + 轻度 Rouge + 卡牌**

## 📁 目录结构
```
assets/           # 资源文件
├── sprites/      # 像素角色/场景图
├── ui/           # UI 元素 (含卡牌界面)
├── audio/        # 音乐/SFX
└── maps/         # 地下城地图数据

scripts/          # 游戏逻辑脚本
docs/             # 设计文档
world/            # 🎨 世界观相关内容
tasks/            # 📝 任务记录与进度跟踪
config/           # 配置文件
cards/            # 🃏 卡牌定义文件 (新增)
```

## 📋 项目规范
- **命名:** snake_case，小写 + 下划线
- **像素尺寸:** 16x16 / 32x32
- **格式:** .json/.lua/.md (文本), .png (图片), .yaml (卡牌数据)
