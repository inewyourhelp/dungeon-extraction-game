# Dungeon Extraction (地下城也要搜打撤)

> 基地建设 × 肉鸽地牢 × 搜打撤 - 全新体验

## 🎮 游戏简介

一款 **2D 像素风** 的地牢探索游戏，融合了 **基地建设**、**肉鸽(Rogue)地牢** 和 **搜打撤** 核心玩法。

玩家在**魔法工业世界**的地表建立和经营自己的 **基地**，招募和培养 **英雄队伍**，学习**斗气**与**魔法**技能，然后深入随机生成的 **地下城（地牢）** 探险搜刮资源，最终带着资源返回地表基地建设升级 —— 形成 **基地→地下城→基地** 的核心循环。

## ⚔️ 世界观：魔法与斗气工业

### 背景设定

这是一个**西方魔幻**与**工业魔法**融合的世界。

数千年前，**斗气**与**魔法**是强者的专属。随着**魔法工业**的兴起，魔法被应用于生产、制造与建设——炼金工厂、附魔机床、魔导蒸汽机……魔法不再是神秘的仪式，而成为推动社会运转的**核心能源**。

如今，旧日地牢中埋藏着失传的魔法秘宝与远古斗气传承。冒险者们组建工会，在酒馆中聚集，**进入地牢搜刮资源、带回基地研究升级** —— 这就是新的生存方式。

### 核心元素

| 元素 | 描述 |
|------|------|
| ⚔️ **斗气** | 战士的核心能量，区别于魔法，更强调身体与意志的修炼 |
| 🔮 **魔法** | 法师的技能，涵盖元素、召唤、炼金等多系分支 |
| ⚙️ **魔法工业** | 魔法驱动的科技树——附魔装备、炼金药剂、魔导装置 |
| 🏰 **工会与酒馆** | 冒险者的聚集地，发布任务、招募英雄、交易资源 |

## 🏰 核心玩法

```
┌─────────────────────────────────────────────┐
│  🏰 基地建设                                 │
│  └─ 酒馆招募英雄 / 建筑升级 / 资源管理 / 职 |
│     业工坊                                  │
│              ↓                               │
│  ⚔️ 地牢探索 (搜打撤)                       │
│  └─ 随机生成关卡 / 战斗 / 搜刮资源            │
│              ↓                               │
│  📦 带着资源返回基地                         │
└─────────────────────────────────────────────┘
```

### 四大核心系统

| 系统 | 描述 |
|------|------|
| 🏰 **基地建设** | 升级建筑、采集资源、打造装备、研究魔法工业 |
| 🍺 **酒馆招募** | 招募英雄、刷新英雄、队伍培养 |
| ⚔️ **职业系统** | 通过职业建筑学习斗气/魔法，决定成长方向 |
| 🌍 **搜打撤** | 进入地牢 → 探索 → 战斗 → 搜刮 → 撤离 |

### 战斗方式

- **回合制策略战斗** - 每场战斗由队伍中的英雄自动触发
- **斗气 + 魔法双体系** - 战士修斗气，法师研魔法，职业决定技能方向
- **Roguelike 体验** - 随机生成的关卡、敌人、事件和奖励
