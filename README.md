# Dungeon Extraction (地下城也要搜打撤)

> 基地建设 × 肉鸽地牢 × 搜打撤 - 全新体验

## 🎮 游戏简介

一款 **2D 像素风** 的地牢探索游戏，融合了 **基地建设**、**肉鸽(Rogue)地牢** 和 **搜打撤** 核心玩法。

玩家在魔法世界中建立和经营自己的 **地下城基地**，招募和培养 **英雄队伍**，然后深入随机生成的地牢探险搜刮资源，最终带着资源返回基地建设升级 —— 形成 **基地→地牢→基地** 的核心循环。

## 🏰 核心玩法

```
┌─────────────────────────────────────────────┐
│  🏰 基地建设                                 │
│  └─ 酒馆招募英雄 / 建筑升级 / 资源管理        │
│              ↓                               │
│  ⚔️ 地牢探索 (搜打撤)                       │
│  └─ 随机生成关卡 / 战斗 / 搜刮资源            │
│              ↓                               │
│  📦 带着资源返回基地                         │
└─────────────────────────────────────────────┘
```

### 三大核心系统

| 系统 | 描述 |
|------|------|
| **🏰 基地建设** | 升级建筑、采集资源、打造装备 |
| **🍺 酒馆招募** | 招募英雄、刷新英雄、队伍培养 |
| **⚔️ 搜打撤** | 进入地牢 → 探索 → 战斗 → 搜刮 → 撤离 |

### 战斗方式

- **回合制策略战斗** - 每场战斗由队伍中的英雄自动触发
- **英雄技能系统** - 每个英雄拥有独特技能和定位
- **Roguelike 体验** - 随机生成的关卡、敌人、事件和奖励
